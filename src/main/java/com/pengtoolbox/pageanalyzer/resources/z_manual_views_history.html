The history view will provide you the following:

  • Table: A table with all your results.
  • Table Filter: Enter a keyword in the filter box to search the table.
  • Eye Icon: Open the result view.
  • Sliders Icon: Open the Gantt Chart.
  • Link Icon: Open the link of the analyzed website in a new tab.
  • Trash Icon: Delete the result.
  • Compare Button: Select two or more results and click the Compare button below the table to get a compare view.
  • Delete Button: Select one or more results and click the Delete button for deleting the results.
